# Solva Product Positioning

## Product Summary

Solva is a lightweight scheduling and client management tool designed specifically for solo service providers.

It replaces the chaotic combination of:

- Instagram DMs
- Phone notes
- Physical planners
- Spreadsheet trackers

with one simple, mobile-first system.

---

# Core Idea

Most scheduling tools are built for:

- Teams
- Salons with receptionists
- Marketplace booking platforms

Solva is built for **one person running their own service business**.

---

# Target Users

Primary users:

- Independent service providers
- Mobile professionals
- Home-based businesses

Examples:

- Nail technicians
- Cleaners
- Massage therapists
- Tutors
- Freelance service providers

Common characteristics:

- Phone-first workflow
- Limited interest in complex software
- Strong focus on client relationships
- Budget conscious

---

# Problem Solved

Solo providers often manage bookings using:

- Instagram messages
- Notes apps
- Paper planners
- Text messages

This leads to:

- Lost bookings
- Missed reminders
- Disorganized client information
- Unprofessional customer experience

---

# Solva Solution

Solva provides a simple system to manage:

- Clients
- Bookings
- Service records
- Invoices

All in one lightweight mobile tool.

Key principles:

- Mobile-first
- Offline capable
- Minimal setup
- Clear ownership of data

---

# Differentiation

| Feature | Solva | Typical Scheduling Tools |
|-------|------|------|
| Mobile-first design | Yes | Often desktop-first |
| Offline support | Yes | Usually requires internet |
| Built for solo operators | Yes | Built for teams |
| Marketplace pressure | None | Often pushes discovery marketplaces |
| Setup complexity | Minimal | Multi-step onboarding |

---

# Product Philosophy

Solva follows three core principles:

### Focus

Only build what solo operators actually need.

### Ownership

Users keep control of their data and client relationships.

### Simplicity

Powerful functionality delivered through calm, intuitive UX.

---

# Long-Term Direction

Solva aims to become a **complete lightweight business system for solo operators**, while remaining small and focused.

Possible expansions:

- Booking pages
- Reminder automation
- Earnings insights
- Client retention tools
- Payment integrations

The goal is not to become a large SaaS platform, but a **trusted tool that small operators rely on daily.**
