# Solva Roadmap

Solva is evolving through small, deliberate releases focused on making life easier for solo service providers.

This roadmap shows the direction of the product and what we're actively working toward.

---

# Current Version

## v1.1 — Prototype

Core functionality introduced:

- Mobile-first scheduling
- Client records and notes
- Configurable services
- Lightweight invoice generation
- Offline-first architecture
- Local-first data storage

The goal of this version was to move from **idea → usable product** and begin testing real workflows.

---

# Next Iteration

## v1.2 — Stabilisation & UX Improvements

Focus: remove friction and strengthen the core workflow.

Planned improvements:

### Faster Setup

- "Create Now" shortcut during setup
- Default services automatically created
- Optional onboarding steps

### Stronger Validation

- Disabled save buttons when forms incomplete
- Inline guidance and helper text
- Guard against invalid workflows

### Mobile UX Improvements

- Floating action buttons on key screens
- Improved thumb reach navigation
- Removal of small top navigation actions

### Invoice Flow Improvements

- Manual invoice creation model
- Prevent duplicate job invoicing
- Business settings applied to invoices

### Trust & Transparency

- Data ownership notice in settings
- Clear explanation of local data storage

# Philosophy

Solva is intentionally small and focused.

It is designed for **solo service providers who run their entire business from their phone.**

That means:

- No unnecessary complexity
- No marketplace commissions
- No bloated dashboards
- Just the tools needed to run a small service business well.
